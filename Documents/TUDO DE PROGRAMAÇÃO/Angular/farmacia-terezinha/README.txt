Para utilizar este sistema:

1* Crie as tabelas em um banco Mysql,
queries localizadas na pasta: backend\queriesdb\

2* Configure o arquivo Config.json na pasta: backend\config\
para conectar com o seu banco de dados

3* Inicie o servidor na pasta backend com o comando: npm start através
do terminal

4* Na pasta frontend, inicie o Ng service.

5* Acesse o localhost:4200 no seu navegador.
